# Clifton Strengths Employee Tool 

## Overview
A Clifton Strengths management tool that builds a knowledge base of employee strength profiles and provides AI-powered communication recommendations to help managers adapt their leadership style to each team member's unique strengths.

## Problem
Managers struggle to remember and effectively utilize each employee's Clifton Strengths profile for personalized communication and leadership approaches

## Target User
Managers and HR administrators who need to optimize team communication and leverage individual employee strengths

## Features
- PDF parser for Clifton Strengths reports
- CSV bulk import with field mapping
- Employee profile search and filtering by strength themes
- Notes system with conversation logging and time-based folders
- AI-powered communication recommendations based on strength profiles
